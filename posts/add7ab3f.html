免费vps

TF HMC2023-07-172023-09-24

你们是不是很想要1台VPS? 现在就有

谷歌云有3个月的试用，有很多节点，非常好，还有300美刀的免费额度。Google:https://cloud.google.com/ （要卡）

AWS就是亚马逊云是一个全球最大的云服务提供商，有免费一年的试用，机器还可以，付费机器有很多人说好，而且还有免费CDN，问题是很容易扣费，AWS：https://aws.amazon.com/ （要卡）

Oracle Cloud
Oracle Cloud 甲骨文在中国有很高知名度，因为给了1H1GG存的VPS（永久），最高可以开到6H26G。有很多地区，列如:日本、美国圣何塞、凤凰城。这个vps很好，机器也很好，流量大，免费。就能注册难，听说200多次注册才可能成功10几次，OC:https://www.oracle.com/cloud/ （要卡）

微软云 能使用免费一年，。免费是 linux 1 和 windows 1 动态 ip，b1s 机型，64G 硬盘存储是免费的 需要信用卡验证或者 GitHub 学生包（学生包不用验证）验证 缺点:和aws一样都容易产生扣费。 微软云:https://azure.microsoft.com/ （要卡，学生不用）

Vultr 需要信用卡验证。注册送 100 美元，一个月时限 试用期限较短，需要信用卡验证 https://www.vultr.com （要卡）

Hax VPS 纯 ipv6，面板可以很方便的添加 ipv4 配置。注册要 tg。永久免费，需要每周手动续订一下 纯 ipv6 需要支持 IPV6 的网络环境 https://hax.co.id （不用卡，续期时间变短）

EU VPS在2022年已经不免费了需要装机费了，总结:总体来说，免费VPS很多，只不多要外币信用卡，不用卡的很少。
