Vercel netlify杂谈

TF HMC2023-06-212023-09-24

Vercel netlify我用vc和nf代替 https://vercel.com https://netlify.com

Vc:vc创始人是 雷吉艾尔莫·劳斯（Guillermo Rauch）出生在阿根廷布宜诺斯艾利斯，是程序员和企业家。参与了开源软件项目的开发跟维护。2007的时候，他移民到美国，并在加州的圣塔芭芭拉市读大学。

大学 Guillermo开始研究 Web 开发领域，他创建了一个 Node.js Web 框架叫 Socket.IO，这个框架在帮助开发者构建实时 Web 应用程序。后面他又创建了另一个 Web 框架叫做 Meteor。在2015年创建了Vercel 公司，推出了Now的静态站点部署服务。Now后来变化成Vercel

Nf:nf创始人有两个人 Mathias Biilmann 和 Christian Bach

Mathias Biilmann 和 Christian Bach 都是成功的技术创业家和企业家。

Mathias Biilmann是web开发者，在丹麦的创业公司中担任首席技术官（CTO），并致力于推动云计算、DevOps 和前端技术等领域的发展。Netlify在建立之前，Mathias Biilmann用过其他托管服务部署静态网站，但是awa，很快啊!他发现这些服务不能满足他的要求，然后他决定自己开发静态网站托管平台，之后诞生了Netlify。
讲述: vc和nf各自都有很多优点，但也有缺点
1.方便:vc和nf都很方便，连接github仓库就能用了
2.流量:vc的流量是每月100G，nf每月也100G限制（但普通站已经够了）
3.使用:有很多项目vc可以部署，vc支持多和架构 nf只支持静态网页但是担建好后 ping值nf更低
4.域名:vc的.app域名被墙了而nf没有 但vc可以邦定自己域名
5.名声:vc技术圈里很多人多用 所以被白嫖烂了，虽说nf名声没有vc好，但是我看上去名声也是不小的

总体上vc和nf各有优缺点
vc不实说真的好 但是对于一些人来说不好 vc注册要手机号验证很烦，而nf不用，只要一会邮箱就行了 但是nf连接github有问题可能要那个…（脑补ing…） 总之静态网站流量又不是很高推荐用nf（hexo…） wp啥的能用vc 随说有些差于nf（速度，高峰期）但是还勉强能接受

https://vercel.com

https://netlify.com
